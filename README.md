# Ecommerce Dashboard project is an admin panel for managing an e-commerce store
It is built with Cascade Code AI-assistant

Current version is available [here](https://ecommerce-eo.netlify.app/)

The live demo uses mock data and product data from open API endpoints on https://fakestoreapi.com/ because it is deployed on a static hosting platform. 

A complete Express + Prisma backend is included in the repository and can be run locally.

## Application uses modern technologies:
- React + TypeScript + Vite
- Tailwind
- TanStack Query
- TanStack Table
- React Hook Form
- Recharts
- Vitest
- React Testing Library
