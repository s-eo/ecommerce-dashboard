# Ecommerce Dashboard project is an admin panel for managing an e-commerce store
It is built with Cascade Code AI-assistant

Application is in active development.
Current version is available [here](https://ecommerce-eo.netlify.app/)

## Application uses modern technologies:
- React + TypeScript + Vite
- Tailwind
- TanStack Query
- TanStack Table
- React Hook Form
- Recharts
- Vitest
- React Testing Library
